На сайте hh.ru можно прочитать отзывы о работодателе, которые собирает сервис dreamjob.ru. 
Публиковать или нет отзыв решают модераторы, есть 8 причин отклонения отзыва. 
Приведена модель, определяющая, будет ли опубликован отзыв и причину отклонения модерации.

Задача NLP мульти-лейбел классификации

Программный стек: pithon, codecs, random, pandas, numpy, copy, pickle, matplotlib, sklearn, keras

Модули:
0_dummy_generater
Функционал: генератор синтетического дата сета для демонстрации (распространение реального датасета запрещено условиями конкурса)
Входные данные: текстовые файлы: 'список профессий.txt', 'voyna-i-mir.txt', 'prestuplenie-i-nakazanie.txt'
Выходные данные: сенерированные датасеты с цифровыми и текстовыми значениями для дальнейшей обработки
HeadHunter_train_dummy.csv – синтетические данные
HeadHunter_test_ dummy.csv – синтетические данные

1_preprocessing_train_table_set_MLP
Функционал: предобработка тренировочного дата сета, включая лемматизацию и присоединение числовых переменных к тексту
Входные данные: HeadHunter_train_dummy.csv – синтетические данные для тренировки модели, words_relation_for_lema.csv – словарь для лемматизации
Выходные данные: file_df_set_train_MLP_new.csv – файл с корпусом текстов, готовый к векторизации и значениями зависимой переменной

2_preprocessing_test_table_set_MLP
Функционал: предобработка тренировочного дата сета, включая лемматизацию и присоединение числовых переменных к тексту
Входные данные: HeadHunter_test_dummy.csv – синтетические данные для тестирования модели, words_relation_for_lema.csv – словарь для лемматизации
Выходные данные: file_df_set_test_MLP_new.csv – файл с корпусом текстов, готовый к векторизации

3_Keras_MLP_train
Функционал: векторизация текста и зависимой переменной, выбор параметров модели по валидационной и стратифицированной тестовой выборках, обучение модели
Водные данные: file_df_set_train_MLP_new.csv – файл с корпусом текстов, готовый к векторизации и значениями зависимой переменной
Выходные данные: TfidfVectorizerMLP.pkl – TF-IDF векторайзер, KerasModel – обученная модель

4_KerasMLP_test
Функционал: векторизация текста тестового дата сета, загрузка данных в модель, получение вектора предсказания зависимой переменной в виде вероятностей, 
            перевод вектора вероятностей в мульти-лейбел вид, требуемый для сабмита
Входные данные: file_df_set_test_MLP_new.csv - файл с корпусом текстов, готовый к векторизации, KerasModel – обученная модель
Выходные данные: submit_MLP.csv – файл с предсказанием в требуемом формате
